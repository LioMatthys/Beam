Open the downloaded APK

Tap it in your notifications or Downloads. Allow “install unknown apps” if asked.

2

Enable Accessibility once

Settings → Accessibility → Beam control → On. This is the phone’s “hands.”

3

Toggle greyed out? (sideload)

Settings → Apps → Beam → ⋮ → Allow restricted settings, then retry step 2.

4

Connect

Open Beam, tap Start sharing, and enter the IP + 6-digit code in the desktop app.
